Publier le programme en ligne

Cette page envoie le programme des activités (page programme.html, sans aucune donnée personnelle) vers votre hébergement, en connexion sécurisée et sortante (FTPS). Votre application et votre base de données ne sont jamais exposées sur internet.

Publication non configurée
Renseignez les identifiants FTP de votre hébergement dans le fichier de configuration du serveur (.env), puis redémarrez l'application :
  • PROGRAMME_FTP_HOST — l'adresse FTP (ex : ftp.monsite.fr)
  • PROGRAMME_FTP_USER / PROGRAMME_FTP_PASSWORD — vos identifiants cPanel
  • PROGRAMME_FTP_DIR — le dossier cible (ex : public_html)
  • PROGRAMME_FTP_FILENAME — le nom du fichier (par défaut programme.html)
  • PROGRAMME_PUBLIC_URL — l'adresse publique finale (facultatif, pour afficher le lien)
{% endif %}

Mise à jour automatique

Pour que la page se mette à jour toute seule, planifiez l'exécution régulière du script tools/publish_programme.py (par exemple chaque nuit), exactement comme la sauvegarde. Vous pouvez aussi cliquer sur « Publier maintenant » à tout moment.

Vous pouvez également télécharger le fichier pour le déposer manuellement.
