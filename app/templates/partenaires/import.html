Importer des partenaires (CSV)

Chargez un fichier CSV pour créer ou mettre à jour des fiches partenaires en masse. L'import reprend tous les champs d'identification. Les partenaires sont rapprochés par leur nom (mise à jour si déjà présent, jamais de doublon).

⬅ Annuaire

1. Récupérer le modèle

Téléchargez le modèle (séparateur « ; », encodage UTF-8), remplissez-le, puis importez-le.

⬇️ Télécharger le modèle CSV

2. Importer le fichier

Colonnes attendues

Seule la colonne nom est obligatoire. Valeurs multiples séparées par « | ».

  • nom, contact_nom, contact_prenom, adresse
  • email_contact, email_general, tel_contact, tel_general
  • secteurs — vos secteurs internes, ex. Numérique|Familles
  • competences — domaines d'orientation (codes) : {{ orientation_domaines.keys()|join(', ') }}
  • territoire_couvert, modalites_orientation, description
  • niveau_orientation — {{ orientation_niveaux.keys()|join(' / ') }}

Astuce : après l'import, lancez « 📍 Géocoder les adresses » sur la carte des partenaires pour les positionner automatiquement.
