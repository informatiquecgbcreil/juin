# P0.3D — Vocabulaire harmonisé

## Termes retenus

| Ancien / technique | Terme utilisateur |
|---|---|
| Subvention | Enveloppe |
| Subvention externe | Enveloppe externe / subvention |
| Fonds propres | Fonds propres / autofinancement |
| Appel à projet | Demande de financement |
| Budget d’appel à projet | Dossier financeur |
| Affectation | Répartition |
| Ligne budgétaire | Type de dépense |
| Compte | Compte budgétaire, surtout en mode expert |

## Principe

On ne renomme pas les modèles Python ni les tables. On harmonise les mots visibles quand les templates sont sûrs, et on documente la règle pour les prochains patchs.
