# P0.3G — Accueil intelligent

## Objectif

Transformer le tableau de bord en vraie rampe de lancement.

L’accueil ne doit pas seulement afficher des chiffres : il doit aider l’utilisateur à choisir une action.

## Ajouts

- Bloc “Je veux faire quoi ?”
- Tuiles métier selon les droits :
  - Finances
  - Activités
  - Publics
  - Bilans
- Résumé rapide :
  - alertes ;
  - sessions ;
  - participants uniques ;
  - consommation budget.
- Variante admin technique :
  - contrôle ;
  - audit navigation ;
  - équipe.

## Principe

On ne supprime pas les actions rapides personnalisables.
On ajoute une couche d’orientation au-dessus pour les utilisateurs qui ne savent pas encore quel module ouvrir.
