# P0.4D — Kiosk : objectifs attendus par session

## Objectif

Le kiosk pédagogique ne doit plus récupérer uniquement les objectifs directement liés à une session.

Il récupère maintenant les objectifs par plusieurs chemins :

- objectifs généraux des projets liés à l’atelier ;
- objectifs spécifiques de l’atelier ;
- objectifs spécifiques enfants des objectifs généraux ;
- objectifs opérationnels liés directement à la session ;
- objectifs opérationnels liés aux modules de la session ;
- objectifs opérationnels liés aux modules de l’atelier ;
- objectifs opérationnels liés aux modules déclarés dans le plan Projet → Atelier → Module ;
- objectifs opérationnels enfants des objectifs spécifiques ;
- objectifs opérationnels rattachés à l’atelier sans session précise.

## Effet attendu

Les objectifs opérationnels créés via un module ou un plan pédagogique remontent désormais dans le kiosk de la séance.

## Bonus P0.4B.1

Le menu “Pédagogie” pointe maintenant vers :

```text
/pedagogie/
```

au lieu de renvoyer directement vers l’ancienne page des référentiels.
